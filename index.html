About Music Player

Music Player is a modern web-based music streaming interface inspired by Spotify. Built using HTML, Tailwind CSS and JavaScript featuring dynamic audio controls, playlist management, and responsive design..

Tech Stack Section

  • • HTML5 Audio API
  • • Tailwind CSS
  • • JavaScript ES6+
  • • Responsive Design

Features Section

  • • Play / Pause / Next / Previous
  • • Shuffle & Repeat
  • • Volume Control
  • • Artist Filtering
  • • Custom Playlist System
